# NativoOne Landing Page

A modern, responsive landing page for NativoOne ERP system built with Next.js, TypeScript, and Tailwind CSS.

## Overview

NativoOne is a comprehensive ERP solution that helps businesses automate processes, simplify fiscal management, and drive growth. This landing page showcases the platform's key features, modules, pricing plans, and customer testimonials.

## Features

- 🎨 Modern UI with Tailwind CSS
- 🔧 Built with Next.js 15 and TypeScript
- 📱 Fully responsive design
- 🎯 SEO optimized
- 🚀 Performance optimized
- 💅 Shadcn UI components

## Tech Stack

- Next.js 15.2.4
- React 19
- TypeScript
- Tailwind CSS
- Radix UI Components
- Lucide React Icons

## Getting Started

### Prerequisites

- Node.js (Latest LTS version recommended)
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone <repository-url>
cd nativo-landing-page
```

2. Install the dependencies:

```bash
npm install
```

3. Running it locally:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
npm run start
```
